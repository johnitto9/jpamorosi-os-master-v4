# Amorosi Labs Agentic Playbook

Sistema de archivos `.md` para guiar a Claude Code en la refacción de `jpamorosi.dev` hacia Amorosi Labs Hall of Fame.

## Uso

Copiar esta carpeta dentro del repo como:

```txt
frontend-app/docs/amorosi-labs/
```

o, si preferís nivel raíz:

```txt
docs/amorosi-labs/
```

Luego abrir Claude Code en la raíz del proyecto y pedir:

```txt
Read docs/amorosi-labs/00_START_HERE.md through docs/amorosi-labs/09_QA_SECURITY_DEPLOY.md. Execute only 05_TASK_FOUNDATION_REFACTOR.md. Stop after build and logs.
```
