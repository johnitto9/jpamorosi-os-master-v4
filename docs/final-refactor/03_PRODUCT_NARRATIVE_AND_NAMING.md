## Narrative rule

The homepage alternates evidence and meaning.

Bad:
`project cards → project cards → project cards → form`

Desired:
`person → proof → machine → systems → intelligence → lab → conversation`

## Copy guardrails

- Avoid self-congratulation.
- Avoid generic startup language.
- Prefer evidence over claims.
- Prefer lived friction over abstract innovation.
- Alún is symbolic context, not a random marketing insert.
- Do not overuse "AI".
