# 10_HOME_STORY_SYSTEM

## Objective

Transform the homepage from a dense sequence of project cards into a narrative rhythm while preserving identity and strong existing sections.

## Canonical order

1. Hero / Intro
2. Before the Systems
3. Proof Rooms
4. You're Inside the Proof
5. Systems in Orbit
6. The Living Layer
7. Lab Fragments
8. Open Channel

## Interlude design system

All three interludes share an aura without becoming copies.

Shared DNA:
- cosmic/noir background;
- restrained cyan/violet accents;
- strong eyebrow;
- one major statement;
- one visual/symbolic focal point;
- one compact explanatory block;
- optional small diagram/chips;
- cinematic reveal;
- reduced-motion fallback.

Do not make them card grids.

## Before the Systems

Purpose:
- humanize;
- create breathing room before proof;
- explain non-linear path;
- introduce Alún symbolically.

Possible visual language:
- train;
- toy-shop;
- industrial night;
- flame;
- machine.

Content topics:
- curiosity before career labels;
- commerce and real-world friction;
- software as instrument;
- Bahía Blanca;
- Alún as thread of meaning.

Keep copy short.

## You're Inside the Proof

Purpose:
- show that `jpamorosi.dev` itself is a live engineered system.

Show:
- public UI;
- Orbe;
- memory;
- Postgres;
- autonomous scout/heartbeat;
- infrastructure;
- key stack.

Reuse project-room-style architecture visuals.

## The Living Layer

Purpose:
- explain loginless continuity and project-building intelligence.

Show:
`arrival → conversation → intent → memory → project DNA → assets → next action`

Mention:
- multiple conversations;
- project rooms;
- branding;
- persistence;
- return/continuation;
- autonomous opportunity layer where appropriate.

## Scroll rhythm

Audit current dead gaps.

Rules:
- preserve cinematic breathing room;
- remove accidental empty vertical corridors;
- project evidence should not be delayed by scroll tax;
- test desktop, mobile, fast scroll and mid-page refresh.

## Acceptance

- canonical order;
- no generic SaaS section;
- interludes related but distinct;
- home no longer feels card-card-card-contact;
- no regressions to side nav, Orbe launcher, locale selector or `/os`;
- reduced-motion path complete.
